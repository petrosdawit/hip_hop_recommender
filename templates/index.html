Spotify Hip Hop Playlist Recommender

This project was designed to incorporate machine learning that I learned from my CS1951-a: Data Science course I took at Brown University. The application can recommend you songs from its 3000+ hip hop song database through unsupervised or supervised learning and create a playlist on Spotify for you. You can edit your created playlists here as well. Click an option to get started!






The supervised learning will take longer time. You will be asked to rate 50 songs with a score from 1-5. You have the option to skip songs if you don't want to rate, but you must rate 50. Once done, the algorithm will create the playlist for you. This option will achieve better results than the unsupervised algorithm. Click the box.




The unsupervised learning will prompt you a search bar where you can add songs you like and have the algorithm fetch songs it thinks you would like and pick and create your playlist. Click the box.
